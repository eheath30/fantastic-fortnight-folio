## Getting started
1. Clone this repository: <br>
    ```https://github.com/eheath30/fantastic-fortnight-folio```
    <br>
2. CD into fantastic-fortnight-folio and install dependencies: <br>
    ```npm i```
    <br>
3. Start the client: <br>
    ```npm start```
